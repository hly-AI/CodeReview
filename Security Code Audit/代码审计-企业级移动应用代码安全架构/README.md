---

## 文档说明

- **主文档**：`../代码审计-企业级移动应用代码安全架构.md`（总纲与结构）
- **本目录**：各章详细内容的扩展版，与主文档一一对应
- **适用对象**：移动开发 Leader、架构师、安全工程师、负责 App 安全与合规的团队成员
- **技术范围**：Android、iOS、KMP（Kotlin Multiplatform）及跨平台应用

## 阅读建议

- **快速入门**：先读主文档总纲，再按需进入 01、02、03、04 章。
- **做审计时**：将 04（思路）、05（基础漏洞）、06（进阶漏洞）、07（二次与供应链）、08（小技巧）作为审计手册配合使用。
- **做设计与规范时**：重点读 09（输入）、10（加密）、11（功能设计）、12（体系建设）。
- **KMP 项目**：在 02、03、04、06、11 中均有 KMP 专门小节，可配合主文档「KMP 架构下的审计侧重」一起使用。
